# Java spring

To get all dependencies run this:

```bash
npm i
```

To run tailwind server

```bash
npm run dev
```

Then run spring by `mvn` or by `VSC` or `IntelliJ`

## Enviromental file

```bash
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DB=
PGADMIN_DEFAULT_EMAIL=
PGADMIN_DEFAULT_PASSWORD=
PG_ADMIN_PORT=
PORT=
HOST=

DB_PLATFORM=
DBMS=

APP_PORT=

DRIVE_CLASS_NAME=
DIALECT_HIBERNATE=

```
